# Mi primer Proyecto

> Este proyecto es para aprender a usar Github.

---

## 📝 Descripción

* **¿Qué problema resuelve?** Automatiza el registro de asistencia para evitar el uso de papel.
* **¿A quién está dirigido?** Estudiantes, desarrolladores independientes, pequeñas empresas.
* **¿Cuál es el propósito principal?** Facilitar la gestión de tareas diarias mediante una interfaz intuitiva.

---

## 🛠️ Tecnologías y Lenguajes Utilizados

Lista de herramientas, librerías y lenguajes clave que implementaste:

* **Lenguaje:** Python.
* **Interfaz/Framework:** Tkinter.
* **Librerías:** Pandas.
* **Control de Versiones:** GitHub.

---

## ⚙️ Instrucciones de Instalación y Ejecución

Sigue estos pasos para clonar y ejecutar el proyecto localmente desde cero:

1. **Clonar el repositorio:**
   git clone https://github.com/mtrxgamer/Mi-primer-proyecto.git

2. **Entrar a la carpeta del proyecto**
cd proyecto

3. **Ejecutar:**
python app.py
